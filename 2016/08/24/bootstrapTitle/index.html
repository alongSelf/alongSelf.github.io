bootstrap(2)排版

  • bootstrap
  • javascript

1.标题

说明

HTML 中的所有标题标签，<h1> 到 <h6> 均可使用。
Bootstrap覆盖了其默认的样式，使其在所有浏览器下显示的效果一样。
bootstrap还提供了 .h1 到 .h6 类，为的是给内联（inline）属性的文本赋予标题的样式。
标题可以包含一个副标题使用<small>

2.段落

说明

段落使用p标签。
同样Bootstrap覆盖了其默认的样式，使其在所有浏览器下显示的效果一样。
段落中使用lead做强调说明

3.强调

说明

强调使用.lead类其他还有
.text-muted：提示，使用浅灰色（#999）
.text-primary：主要，使用蓝色（#428bca）
.text-success：成功，使用浅绿色(#3c763d)
.text-info：通知信息，使用浅蓝色（#31708f）
.text-warning：警告，使用黄色（#8a6d3b）
.text-danger：危险，使用褐色（#a94442）

4.粗体

说明

普通的元素中我们一般通过font-weight设置为bold关键词给文本加粗。
在Bootstrap中，可以使用和标签让文本直接加粗。

5.斜体

说明

在Bootstrap通过使用标签<em>或<i>来实现。

6.对齐

说明

类 说明
.text-left 设定文本左对齐
.text-center 设定文本居中对齐
.text-right 设定文本右对齐
.text-right 设定文本右对齐
.text-justify 设定文本对齐,段落中超出屏幕部分文字自动换行
.text-nowrap 段落中超出屏幕部分不换行
.text-lowercase 设定文本小写
.text-uppercase 设定文本大写
.text-capitalize 设定单词首字母大写

7.列表

说明

bootstrap的列表总共有6种类

1.有序列表

2.无序列表

3.去点列表

4.内联列表

5.定义列表

6.定义列表

8.代码

说明

1.使用<code></code>来显示单行内联代码
2.使用<pre></pre>来显示多行块代码
3.使用<kbd></kbd>来显示用户输入代码
4..pre-scrollable，就可以控制代码块区域

9.代码

说明

类 说明
.table-striped 斑马线表格
.table-bordered 带边框的表格
.table-hover 鼠标悬停高亮的表格
.table-condensed 紧凑型表格
.table-responsive 响应式表格
